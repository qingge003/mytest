
IDEA快捷键:

soutv    生成打印语句并输出变量
iter     增强型 for 循环
ifn      生成 if xx 为 null

shift + 上下箭头 : 快速选择
alt+shift+down : 向下移动行
alt+shift+up : 向上移动行

Ctrl + 左键单击 : 进入光标所在的方法/变量的接口或是定义处
alt + 鼠标左键按住往下拖动 : 一次编辑多行

F2	跳转到下一个高亮错误 或 警告位置 （必备）
shift + F6:批量修改变量名
Shift + F6	对文件 / 文件夹 重命名

ctrl + H :  查看类的继承结构
ctrl + P :  方法参数提示显示
Ctrl + N   : 根据输入的 类名 查找类文件
Ctrl + F   : 在当前打开文件中搜索
Ctrl + R     在当前文件进行文本替换 
Ctrl + E	显示最近打开的文件记录列表
Ctrl + U	前往当前光标所在的方法的父类的方法 / 接口定义
Ctrl + F12  : 当前类中查看所有方法
Ctrl + shift + F12    : 窗口放大/缩小

alt + 左右箭头  : 切换代码左右窗口
alt + 上下箭头 : 浏览当前窗口方法
alt + 数字  :显示对应数值的选项卡



Ctrl + Alt + L	格式化代码，可以对当前文件和整个包目录使用 （必备）
Ctrl + Alt + O	优化导入的类，可以对当前文件和整个包目录使用 （必备）
Ctrl + Alt + Y	同步、刷新
Ctrl + Shift + U	对选中的代码进行大 / 小写轮流转换 （必备）
Ctrl + Shift + J	自动将下一行合并到当前行末尾 （必备）
Ctrl + Shift + C        复制当前文件磁盘路径到剪贴板

eclipse快捷键:

alt +/           :     打开代码辅助菜单
alt +?           :     显示自动提示功能
ctrl + 1         :     红线给出解决方案
ctrl + 1 (选择splite) : 分离变量的声明和赋值
F2：重命名工程名，文件名
F3: 转到定义，将光标放在类后接口或方法上，查看其定义

alt + enter   :    打开选中文件属性
alt+ 左右箭头 :    返回上一次光标停留位置，即使在不同文件中。
alt +上下箭头 :    移动光标所在一行
ctrl + R      :    批量修改同名变量
ctrl + E      :    打开编辑窗口查看目录,按一次下翻一个
ctrl + D      :    删除光标所在行
ctrl + T      ：   查看类的继承，实现关系
ctrl + O      ：   查看类的结构
ctrl + Page Up       :     切换到当前编辑窗口的上一个窗口
ctrl + Page Down     :     切换到当前编辑窗口的下一个窗口
ctrl + alt +上箭头   :     向上复制
ctrl + alt +下箭头   :     向下复制
ctrl+shift+O         ：    导包
ctrl+shift+F         ：    格式化代码(对代码重新排版)
ctrl+shift+T         :     打开查看类窗口 (输入想查看类的名字)
ctrl+shift+R         :     打开查看资源窗口 (输入想查看类的名字)
shift + enter        :     在光标所在行下添加空行

alt+shift+S    ： 弹出source菜单(构造函数)
alt+shift+Z    :  对选中代码段进行surrount with。(try catch,for,while)
alt+shift+R    :  变量批量更名
alt+shift+M    :  抽取方法(将选中代码抽取成方法)
alt+shift+L    ： 增加抽取变量
alt+shift+A    ： 块编辑
alt+shift+J    :  添加注释
